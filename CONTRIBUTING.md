# Contributing to ChineseCharacterCounter

Thank you for your interest in contributing!

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/ChineseCharacterCounter.git`
3. **Create a branch**: `git checkout -b feature/your-feature-name`
4. **Make your changes** and commit: `git commit -m "Add: brief description"`
5. **Push to your fork**: `git push origin feature/your-feature-name`
6. **Open a Pull Request**

## Code Style

- Use TypeScript
- Run `npm test` before submitting PR
- Keep code consistent with existing style

## Questions / Issues

- For bugs, open an issue with the bug report template
- For features, open an issue with the feature request template
- For questions, feel free to open a discussion
